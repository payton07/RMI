RMI

This repository contains a project implemented in Java.

Overview

RMI most likely refers to Remote Method Invocation, a Java API that allows objects to communicate remotely. This project demonstrates or implements concepts related to Java RMI.

Features

100% Java implementation
Example of remote communication using Java RMI
Getting Started

Prerequisites

Java Development Kit (JDK) 8 or higher
Cloning the Repository

bash
git clone https://github.com/payton07/RMI.git
cd RMI
Running

Instructions for building and running will depend on your implementation. Typically, Java RMI projects involve:

Compile the Java files:
bash
javac *.java
Start the RMI Registry:
bash
rmiregistry
Run the server and client classes as needed.
